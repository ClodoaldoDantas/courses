## 💻 Projeto
Template para um dashboard de gerenciamento de cursos online.

## 🚀 Tecnologias

- [React](https://pt-br.reactjs.org/)
- [Styled Components](https://styled-components.com/)
- [TypeScript](https://www.typescriptlang.org/)

## 🔥 Instalação

```bash
npm install
npm run dev
```
